# LL(1) Parser Generator Web Application

## 🚀 Features
- FIRST & FOLLOW computation
- LL(1) parsing table generation
- Stack-based parsing
- Parse tree visualization (Graphviz)
- Dynamic grammar input
- Docker support

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- Graphviz
- Docker

## 🐳 Run with Docker

```bash
docker build -t ll1-parser .
docker run -p 5000:5000 ll1-parser
